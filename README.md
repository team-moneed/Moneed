## 프로젝트 구조

```
Moneed/
├── apps/
│   ├── web/                    # Next.js 웹 애플리케이션
│   └── proxy-server/           # Express 프록시 서버
├── packages/
│   ├── shared-types/          # 공유 타입 정의
│   └── shared-utils/          # 공유 유틸리티
├── prisma/                    # 데이터베이스 스키마
└── public/                    # 정적 자산
```
