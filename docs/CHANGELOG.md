## [1.0.0] - 2025-01-XX

### 추가됨

#### 🏗️ 아키텍처 개선

- **FSD 아키텍처 도입**: Feature-Sliced Design 패턴을 적용한 전체 프로젝트 구조 개편
- **레이아웃 컴포넌트 표준화**: 모든 페이지에 일관된 레이아웃 컴포넌트 시스템 구축
- **모달 시스템 표준화**: 모달 사용 패턴을 통일하여 일관성 개선
- **새로운 기능 컴포넌트**: write-comment 기능 모듈 추가

#### 🔧 개발 환경 개선

- **토큰 관리 시스템 개선**: TokenManager와 TokenUtils 통합으로 중복 로직 제거
- **컴포넌트 구조 개선**: 기능별 디렉토리 구조로 재정리하여 유지보수성 향상
- **유틸리티 함수 추가**: 다양한 공통 유틸리티 함수들을 shared 레이어로 이동

#### 📱 UI/UX 개선

- **네비게이션 아이콘 개선**: 로그인 상태에 따른 네비게이션 아이콘 표시 처리
- **페이지별 레이아웃 설정**: 마이페이지, 온보딩, 웰컴 페이지 등 전용 레이아웃 구현
- **모바일 최적화**: 모바일 환경에서의 사용자 경험 개선

### 변경됨

#### 🔄 구조적 변경

- **디렉토리 구조 대규모 리팩토링**:
    - `src/app/` → `src/screens/` 구조로 변경
    - `src/components/` → `src/shared/ui/` 구조로 통합
    - `src/apis/` → `src/features/*/api/` 구조로 분산
- **인증 기능 통합**: auth 기능을 user 기능으로 통합하여 도메인 중심 구조로 개편
- **페이지 컴포넌트 구조 변경**: Next.js App Router 패턴에 맞춘 컴포넌트 구조 개선

#### 🎨 컴포넌트 개선

- **버튼 컴포넌트 표준화**: Button, GoBackButton, NextButton 등 표준화된 버튼 컴포넌트
- **링크 컴포넌트 체계화**: ButtonLink, IconLink, LogoLink 등 역할별 링크 컴포넌트
- **레이아웃 컴포넌트 분리**: BottomNav, RootNav, SubNav 등 역할별 네비게이션 컴포넌트

#### 🔧 개발 도구 개선

- **상태 관리 개선**: Zustand 스토어 구조 개선 및 타입 안전성 강화
- **API 클라이언트 개선**: 서버/클라이언트 분리된 API 클라이언트 구조
- **유틸리티 함수 통합**: 공통 유틸리티 함수들을 shared 레이어로 이동

### 제거됨

#### 🐳 Docker 환경 제거

- **Docker 관련 파일 완전 제거**:
    - `Dockerfile`, `docker-compose.yml`, `.dockerignore` 삭제
    - `nginx/` 설정 파일들 삭제
    - `env-templates/` 디렉토리 전체 삭제
- **배포 스크립트 제거**:
    - `scripts/build-docker.sh`, `scripts/deploy-ec2.sh` 등 Docker 배포 관련 스크립트
    - `scripts/setup-dev.ps1`, `scripts/setup-dev.sh` 등 개발 환경 설정 스크립트
- **문서 정리**:
    - `docs/DEPLOYMENT.md`, `docs/monorepo_plan.md` 삭제

#### 🔄 CI/CD 파이프라인 단순화

- **GitHub Actions 워크플로우 단순화**:
    - `.github/workflows/ci.yml` 완전 제거
    - `deploy.yml`에서 Docker 빌드/배포 단계 제거
    - 복잡한 테스트 파이프라인 제거
    - 기본적인 CI 기능만 유지

#### 📦 불필요한 파일 정리

- **중복 컴포넌트 제거**: 기존 레이아웃 컴포넌트들의 중복 제거
- **사용하지 않는 유틸리티 제거**: 더 이상 사용하지 않는 유틸리티 함수들 정리
- **타입 정의 정리**: 중복되거나 사용하지 않는 타입 정의들 제거

### 수정됨

#### 🐛 버그 수정

- **빌드 에러 수정**: TypeScript 컴파일 에러 및 import 경로 에러 해결
- **파일 이름 수정**: `comment.controller..ts` → `comment.controller.ts` 파일명 정정
- **리다이렉트 경로 수정**: 글쓰기 완료 후 리다이렉트 되는 페이지 경로 수정
- **import 에러 수정**: 모듈 import 경로 에러들 수정

#### 🔧 성능 개선

- **번들 크기 최적화**: 불필요한 코드 제거로 번들 크기 감소
- **컴포넌트 최적화**: 재렌더링 최적화 및 메모이제이션 적용
- **API 호출 최적화**: 중복 API 호출 제거 및 캐싱 개선

#### 🎯 사용자 경험 개선

- **로딩 상태 개선**: 페이지 전환 시 로딩 상태 표시 개선
- **에러 처리 강화**: 사용자 친화적인 에러 메시지 및 처리 로직 개선
- **접근성 개선**: 웹 접근성 가이드라인에 맞춘 UI 컴포넌트 개선

### 기술적 개선

#### 🏗️ 아키텍처

- **FSD 패턴 적용**: Feature-Sliced Design 아키텍처로 전체 프로젝트 구조 개편
